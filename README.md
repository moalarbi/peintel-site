# Peintel - Services de Rénovation et Peinture

موقع تعريفي احترافي لشركة Peintel للتجديد والطلاء، مع دعم كامل لثلاث لغات (الفرنسية والإنجليزية والعربية).

## الميزات الرئيسية

- **دعم متعدد اللغات:** الفرنسية (FR)، الإنجليزية (EN)، والعربية (AR)
- **نظام حجز عبر واتساب:** نموذج حجز متكامل يرسل الطلبات مباشرة إلى واتساب
- **تصميم حديث:** واجهة احترافية مع Tailwind CSS و Framer Motion
- **متجاوب بالكامل:** يعمل بشكل مثالي على جميع الأجهزة
- **7 خدمات متخصصة:** عرض شامل لجميع الخدمات المقدمة
- **صفحات منفصلة:** صفحات مخصصة لكل خدمة والمشاريع والاتصال

## المتطلبات

- Node.js 22+
- pnpm 10.4.1+

## التثبيت والتطوير

```bash
# تثبيت المكتبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# بناء المشروع
pnpm build

# عرض معاينة الإنتاج
pnpm preview
```

## البناء والنشر على GitHub Pages

### الخطوة 1: إعداد المستودع

```bash
git clone https://github.com/moalarbi/peintel-site.git
cd peintel-site
pnpm install
```

### الخطوة 2: البناء

```bash
pnpm build
```

سيتم إنشاء مجلد `dist/public` يحتوي على الملفات الثابتة الجاهزة للنشر.

### الخطوة 3: النشر على GitHub Pages

يتم النشر تلقائياً عند دفع التغييرات إلى فرع `main` من خلال GitHub Actions.

للنشر اليدوي:

```bash
# تثبيت gh-pages
npm install --save-dev gh-pages

# نشر المشروع
npx gh-pages -d dist/public
```

## الصفحات الرئيسية

- **الصفحة الرئيسية** (`/fr`, `/en`, `/ar`): عرض شامل للخدمات والمشاريع
- **صفحات الخدمات** (`/:lang/service/:serviceId`): تفاصيل كل خدمة مع FAQ
- **صفحة المشاريع** (`/:lang/projects`): معرض المشاريع المنجزة
- **صفحة الاتصال** (`/:lang/contact`): معلومات الاتصال ونموذج الحجز
